Bot hecho en BDFD el cual está inspirado en Ballsdex. Sistema automático de spawn por mensajes en los chats, coleccionar raiderballs, visualizar las estadísticas de cartas, guardado de raiderballs, economía, tienda, personalización de perfil y mucho más.

## Características del bot
- Sistema de spawn automático de balls.
- Sistema de captura y guardado de balls.
- Sistema de probabilidades especiales de spawn de balls.
- Sistema de visualización de estadísticas/cartas de la ball capturada.
- Sistema de economía global.
- Sistema de tienda y compra.
- Personalización del perfil de jugador.
- Comandos exclusivos para el administrador del bot.

## ¿Qué requiere el bot para funcionar?
- Desarrollarlo en BDFD utilizando BDScript 2.
- Creación de algunas variables las cuales [se especifican aquí](https://github.com/deozrammy/BDFD-Raidersdex/blob/main/VARIABLES.md)

- Activar `Message Content Intents` en el portal de desarrolladores de discord.
- Verificar que posea los permisos de rol necesarios dentro de tu servidor para que este funcione.

## Posibles bugs y edición del código
El bot en cuestión no debería de presentar ningún problema en su código, simplemente hay que ser cuidadoso y observar bien lo que es el código y el texto sin riesgo.
